# 🍰 SweetTreats Bakery Website

**A responsive front-end website developed for the fictional SweetTreats Bakery, showcasing baked goods, custom cakes, and bakery services while demonstrating the fundamentals of modern web development using HTML, CSS, and JavaScript.**

---

## 📋 Project Overview

The **SweetTreats Bakery Website** is my **first web development project**, developed as part of the **WEDE5020 Web Development** module. The project was created to apply the fundamental concepts of front-end web development by designing and building a complete website for a fictional bakery business.

The website serves as an online presence for **SweetTreats Bakery**, allowing visitors to explore the bakery's products, learn about its story, browse featured items, and access contact information through a clean, responsive, and user-friendly interface.

Throughout the project, I focused on building a solid understanding of website structure, styling, responsiveness, and basic client-side interactivity while following good design principles.

### 🎯 Project Objectives

* Develop a fully responsive multi-page website.
* Demonstrate semantic HTML for well-structured web pages.
* Apply CSS to create an attractive and consistent visual design.
* Use JavaScript to add interactivity and improve user experience.
* Practice website navigation and page layout design.
* Strengthen foundational front-end web development skills.

### 👥 Target Audience

The website is intended for users who want to:

* Browse freshly baked products.
* Explore custom cakes and pastries.
* Learn more about the fictional SweetTreats Bakery.
* View bakery services and featured products.
* Contact the bakery for enquiries or future orders.
