# SweetTreats Bakery Website  

This project is a **POE/Assignment** for the **Web Development course (WEDE5020)**. The purpose of this website is to practice and demonstrate the **fundamentals of web development** using:  

- **HTML** for structure  
- **CSS** for styling and layout  
- **JavaScript** for interactivity  

The website is designed for a fictional bakery called **SweetTreats Bakery**. It serves as a practical learning exercise to apply the basics of front-end web development, including **page structure, navigation, responsive design, and simple scripting**.
